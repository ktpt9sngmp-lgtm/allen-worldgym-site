公開方式 A｜Netlify Drop
1. 將 allen-worldgym-site.zip 解壓縮。
2. 開啟 Netlify Drop 網頁。
3. 把 allen-worldgym-site 資料夾拖進頁面。
4. 系統會產生可公開分享的網址。

公開方式 B｜GitHub Pages
1. 建立一個新的公開 GitHub repository。
2. 上傳 index.html 到 repository 根目錄。
3. 到 Settings → Pages。
4. Source 選 Deploy from a branch，Branch 選 main / root。
5. 儲存後即可取得公開網址。
